Teaching

I teach C Language, Python Language, Data Structures, Design and Analysis of Algorithms, Pattern Recognition using Deep Learning, and Computer Vision.

Course: C Language

This course builds a strong foundation in structured programming using C. Topics include:

  • Fundamentals — data types, variables, operators, expressions, and input/output
  • Control Flow — branching (if-else, switch) and loops (for, while, do-while)
  • Functions — parameter passing, scope and storage classes, recursion
  • Arrays and Strings — one- and multi-dimensional arrays, string handling
  • Pointers — pointer arithmetic, pointers with arrays and functions, dynamic memory allocation
  • Structures and Unions — user-defined types, nested structures, typedef
  • File Handling — text and binary files, file operations
Course: Python Language

This course covers programming in Python from the basics to practical data handling. Topics include:

  • Fundamentals — syntax, data types, operators, and control flow
  • Data Structures — lists, tuples, dictionaries, sets, and comprehensions
  • Functions and Modules — arguments, lambda functions, packages, and the standard library
  • Object-Oriented Programming — classes, inheritance, and polymorphism
  • File Handling and Exceptions — reading/writing files, error handling
  • Scientific Python — NumPy arrays, Pandas dataframes, and plotting with Matplotlib
Course: Data Structures

This course covers the organization and manipulation of data for efficient computation. Topics include:

  • Linear Structures — arrays, linked lists (singly, doubly, circular), stacks, and queues
  • Trees — binary trees, binary search trees, AVL trees, heaps, and tree traversals
  • Graphs — representations, BFS and DFS, spanning trees, shortest paths
  • Hashing — hash functions, collision resolution, hash tables
  • Searching and Sorting — linear and binary search; bubble, insertion, selection, merge, quick, and heap sort
  • Applications — expression evaluation, priority queues, and memory management
Course: Design and Analysis of Algorithms

This course covers fundamental techniques for designing efficient algorithms and analyzing their complexity. Topics include:

  • Asymptotic Analysis — growth of functions; Big-O, Big-Theta, and Big-Omega notations; best-, average-, and worst-case analysis; complexity classes P, NP, NP-Complete, and NP-Hard
  • Brute Force and Exhaustive Search — linear search, naive string matching, 0/1 Knapsack, Travelling Salesman Problem
  • Divide and Conquer — recurrence relations (substitution, recursion tree, Master Theorem); Merge Sort, Quick Sort, Binary Search, Strassen’s algorithm
  • Greedy Algorithms — greedy-choice property and optimal substructure; Activity Selection, Huffman Coding, Prim’s and Kruskal’s algorithms
  • Dynamic Programming — overlapping subproblems and memoization; Longest Common Subsequence, Matrix Chain Multiplication, 0/1 Knapsack
  • Graph Algorithms — BFS and DFS, shortest paths (Dijkstra, Bellman-Ford), minimum spanning trees, topological sorting
  • Backtracking and Branch & Bound — N-Queens, graph coloring, subset sum
Course: Pattern Recognition using Deep Learning

This course introduces the foundations of statistical pattern recognition and modern deep learning. Topics include:

  • Foundations of Pattern Recognition — Bayes decision theory, parametric and non-parametric estimation, discriminant functions
  • Feature Extraction and Dimensionality Reduction — PCA, LDA, feature selection strategies
  • Classical Classifiers — k-Nearest Neighbors, Support Vector Machines, decision trees, ensemble methods
  • Neural Networks — perceptrons, multilayer networks, backpropagation, regularization, and optimization
  • Convolutional Neural Networks — architectures for image classification, detection, and segmentation; transfer learning
  • Sequence Models and Transformers — recurrent networks, attention mechanisms, transformer architectures
  • Applications — medical image analysis, object detection, and generative models
Course: Computer Vision

This course covers how machines extract meaning from images and video. Topics include:

  • Image Formation and Processing — sampling and quantization, color spaces, histogram processing, filtering in spatial and frequency domains
  • Edges and Features — edge detection (Sobel, Canny), corners, SIFT/ORB descriptors, feature matching
  • Segmentation — thresholding, region-based methods, clustering, and morphological operations
  • Object Detection and Recognition — classical pipelines (HOG + SVM) and modern detectors (YOLO family)
  • Deep Learning for Vision — CNN architectures, transfer learning, and vision transformers
  • Applications with OpenCV — face recognition, medical image analysis, and video analytics
